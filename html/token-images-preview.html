  • Automatic Image Assignment: When you create tokens with symbols TS, MST, or LTS, the system automatically assigns the corresponding image
  • Metadata Integration: Images are embedded in token metadata using the Metaplex Token Metadata program
  • Wallet Display: Backpack wallet will automatically display these images when showing your tokens
  • Custom Tokens: Custom tokens without predefined images will still have proper metadata, just without an image
